atto — WebDAV Back-Port

Started 05-Jul-2026. The WebDAV read-and-write support that grew up in the Yioop project's copy of WebSite — where it backs the self-hosted git wiki pages that clients clone, pull, and push over dumb WebDAV — is brought back into atto's own WebSite so the framework ships it directly, with an example that shows it and a reworked final example that demonstrates extending an existing server rather than the raw Fiber primitives.

The WebDAV verbs (OPTIONS, PROPFIND, MKCOL, PUT, GET, DELETE, MOVE, and the lock verbs where present) and their routing should live in atto's own WebSite, decoupled from anything Yioop-specific, keeping atto's no-config, single-file character.
